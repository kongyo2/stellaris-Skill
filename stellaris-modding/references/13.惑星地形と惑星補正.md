# 惑星地形（Deposits）と惑星補正（Planet Modifiers）

惑星の特徴や障害物、軌道上の資源、および惑星生成時に付与される補正を定義するためのリファレンス。

---

## 惑星地形（Deposits）

惑星地形は惑星上のオブジェクトで、独自の資源生産・維持費と惑星への補正を持つことができる。`common/deposits/` フォルダで定義する。

### 地形の種類

| 種類 | 対象惑星 | `is_for_colonizable` | 特徴 |
|------|----------|---------------------|------|
| **惑星特徴（Planetary Features）** | 入植可能惑星 | `yes` | プレイヤーが確認可能、通常は有益で除去不可 |
| **障害物（Blockers）** | 入植可能惑星 | `yes` | 有害、除去可能（リソース消費）、`deposit_cat_blockers`カテゴリ |
| **軌道地形（Orbital Deposits）** | 入植不可惑星 | `no` | 採掘・研究ステーションで収集 |

---

### データ構造

```pdx
deposit_name = {
    # 基本設定
    is_for_colonizable = yes/no        # 入植可能惑星用か（デフォルト: no）
    is_null = yes                      # ヌル地形として設定
    
    # アイコンとカテゴリ
    icon = <deposit_key>               # 別の地形のアイコンを使用
    category = deposit_cat_xxx         # 地形カテゴリ
    
    # 軌道ステーション（軌道地形のみ）
    station = shipclass_mining_station   # または shipclass_research_station
    
    # 表示設定
    use_for_min_max_adjustments = yes  # 惑星ビューの区域上限更新に使用
    
    # 補正（planet scope）
    planet_modifier = { }              # 惑星への常時補正
    triggered_planet_modifier = {      # 条件付き惑星補正
        potential = { <conditions> }
        <modifiers>
    }
    country_modifier = { }             # 所有帝国への補正
    blocked_modifier = { }             # 障害物の後ろにある間の補正（未使用）
    
    # リソース（Economy Unit）
    resources = {
        category = <economy_category>  # planet_deposits / deposit_blockers / orbital_xxx_deposits
        produces = { <resource> = <amount> }
        cost = { <resource> = <amount> }  # 障害物除去コスト
    }
    
    # 出現条件（planet scope）
    potential = { <conditions> }       # スポーン可能条件
    drop_weight = {                    # スポーン確率
        weight = <int>
        modifier = { factor = <float> <conditions> }
    }
    
    # 障害物専用
    time = <int>                       # 除去に要する日数
    prerequisites = { <tech_key> }     # 除去に必要な技術
    can_be_cleared = { <conditions> }  # 除去可能条件
    on_cleared = { <effects> }         # 除去時の効果
    
    # テラフォーミング時のスワップ
    should_swap_deposit_on_terraforming = yes
    terraforming_swap_types = { d_xxx d_yyy }
    all_terraforming_swap_types = yes
    use_weights_for_terraforming_swap_types = yes
    
    # 障害物のスワップタイプ（障害物除去時に生成される地形）
    blocker_swap_types = { d_xxx }
    all_blocker_swap_types = yes
    use_weights_for_blocker_swap_types = yes
    
    # AI評価
    ai_weight = { }                    # AI障害物除去優先度
}
```

### 地形カテゴリ

`common/deposit_categories/` で定義。惑星の区域分布を多様化するために使用。

```pdx
deposit_cat_food = { }
deposit_cat_energy = { }
deposit_cat_minerals = { }
deposit_cat_rare = { 
    important = yes     # 惑星ビューで希少地形として表示
}

deposit_cat_blockers = { 
    blocker = yes       # 障害物としてカウント
}
deposit_cat_blockers_natural = {
    blocker = yes
}
deposit_cat_rare_blocker = { 
    blocker = yes
    important = yes
}
```

### 例

#### 惑星特徴（発電区域+1）
```pdx
d_hot_springs = {
    is_for_colonizable = yes
    use_for_min_max_adjustments = yes
    category = deposit_cat_energy
    
    should_swap_deposit_on_terraforming = yes
    use_weights_for_terraforming_swap_types = yes
    terraforming_swap_types = { d_arid_highlands d_buzzing_plains }
    
    planet_modifier = { 
        district_generator_max = 1
    }
    
    potential = {
        OR = {
            is_cold = yes
            is_wet = yes
            is_planet_class = pc_nuked
        }
    }
    
    drop_weight = {
        weight = @high
        modifier = {
            factor = @planet_type_bonus
            is_dry = yes
        }
    }
}
```

#### 障害物（危険な野生動物）
```pdx
d_dangerous_wildlife_blocker = {
    time = 270
    is_for_colonizable = yes
    category = deposit_cat_blockers_natural
    
    can_be_cleared = {
        custom_tooltip = {
            fail_text = has_ranger_lodge
            planet = { NOT = { has_building = building_ranger_lodge } }
        }
    }
    
    resources = {
        category = deposit_blockers_natural
        cost = { energy = 500 }
    }
    
    potential = {
        planet_size >= 10
        NOT = { is_planet_class = pc_volcanic }
    }
    
    planet_modifier = { planet_max_districts_add = -2 }
    
    triggered_planet_modifier = {
        potential = { biologist_is_ranger_trigger = yes }
        planet_biologists_society_research_produces_add = @ranger_large_production
    }
    
    on_cleared = {
        hidden_effect = {
            if = {
                limit = { exists = owner owner = { has_civic = civic_hive_devouring_swarm } }
                owner = { add_resource = { food = 200 } }
            }
        }
    }
    
    drop_weight = {
        weight = @blocker_low
        modifier = { factor = 0.5 planet_size < 15 }
        modifier = { factor = 0 num_free_districts = { type = district_city value < 2 } }
    }
    
    prerequisites = { "tech_dangerous_wildlife" }
}
```

#### 軌道地形（社会学研究）
```pdx
d_society_1 = {
    resources = {
        category = orbital_research_deposits
        produces = { society_research = 1 }
    }
    station = shipclass_research_station 
    is_for_colonizable = no
    
    potential = { is_planet_class = pc_toxic }
    drop_weight = { weight = 0 }
}
```

### ヌル軌道地形

地形のない入植不可惑星のプレースホルダー。

```pdx
d_null_deposit = {
    is_null = yes
    
    potential = {
        is_primary_star = no
        NOR = {
            is_planet_class = pc_ringworld_habitable_damaged
            is_planet_class = pc_ringworld_tech_damaged
        }
    }
    
    drop_weight = { weight = 100 }
}
```

## 惑星補正（Planet Modifiers）

惑星生成時に付与される追加の補正。`common/planet_modifiers/` フォルダで定義。静的補正（static modifiers）にリンクするが、同一ではない。

> [!NOTE]
> スクリプトやアノマリーで直接追加される静的補正は、ここでの定義は不要。

### データ構造

```pdx
pm_modifier_name = {
    is_null = yes                      # ヌル補正（静的補正なし）
    
    potential = { <conditions> }       # 出現可能条件（planet scope）
    
    spawn_chance = {
        base = <int>                   # 基本確率（デフォルト: 0）
        modifier = {
            add = <int>                # 加算
            <conditions>
        }
        modifier = {
            factor = <float>           # 乗算
            <conditions>
        }
    }
    
    modifier = "<static_modifier_key>" # リンクする静的補正のキー
}
```

---

### ヌル惑星補正

他の補正が選ばれない場合のプレースホルダー。複数の補正を持つ惑星を稀にするために使用。

```pdx
pm_null = {
    is_null = yes
    
    spawn_chance = {
        base = 275                     # 高い基本確率
        modifier = {
            factor = 3
            num_modifiers > 2          # 既存補正が多いほど選ばれやすい
        }
        modifier = {
            factor = 3
            num_modifiers > 1
        }
        modifier = {
            factor = 3
            num_modifiers > 0
        }
    }
}
```

### 例

#### 危険な野生動物
```pdx
pm_dangerous_wildlife = {
    spawn_chance = {
        modifier = { add = 10 is_planet_class = "pc_desert" }
        modifier = { add = 10 is_planet_class = "pc_tropical" }
        modifier = { add = 10 is_planet_class = "pc_continental" }
        # 他の惑星タイプにも同様に add = 10
        modifier = { factor = 2 has_planet_modifier = "pm_lush" }
        modifier = { factor = 2 has_planet_modifier = "pm_high_gravity" }
        modifier = { factor = 0 has_planet_modifier = "pm_dangerous_wildlife" }
        modifier = { factor = 0 has_planet_modifier = "pm_docile_fauna" }
        modifier = { factor = 0 num_modifiers > 1 }
    }
    
    modifier = "dangerous_wildlife"
}
```

#### 高重力
```pdx
pm_high_gravity = {
    spawn_chance = {
        modifier = { add = 10 is_planet_class = "pc_desert" }
        # 各惑星タイプに add
        modifier = { factor = 2 planet_size > 19 }
        modifier = { factor = 2 planet_size > 22 }
        modifier = { factor = 0.5 planet_size < 16 }
        modifier = { factor = 0 planet_size < 12 }
        modifier = { factor = 0 has_planet_modifier = "pm_low_gravity" }
        modifier = { factor = 0 has_planet_modifier = "pm_high_gravity" }
        modifier = { factor = 0 num_modifiers > 1 }
    }
    
    modifier = "high_gravity"
}
```

## 関連エフェクト

### 地形操作
```pdx
add_deposit = d_xxx                                  # 地形を追加
remove_deposit = d_xxx                               # 地形を除去
clear_deposits = yes                                 # 全地形をクリア
clear_blocker = yes                                  # 障害物をクリア
```

### 惑星補正操作
```pdx
add_modifier = { modifier = pm_xxx days = -1 }       # 惑星補正を追加（永続）
remove_modifier = pm_xxx                             # 惑星補正を除去
```

---

## 関連条件

### 地形チェック
```pdx
has_deposit = d_xxx                                  # 特定の地形を持つか
num_deposits = { type = d_xxx value >= 1 }           # 地形の数
any_deposit = { <conditions> }                       # いずれかの地形が条件を満たすか
```

### 惑星補正チェック
```pdx
has_planet_modifier = pm_xxx                         # 特定の惑星補正を持つか
num_modifiers > 1                                    # 補正の数（spawn_chance内で使用）
```
