# スコープ (Scope)

スコープはトリガーやエフェクトでコンテキストを切り替える（リンクする）機構。ドットスコーピング（例: `root.owner.capital_scope`）も可能。

## 基本スコープ

| スコープ | 説明 | 入力 | 出力 |
|---------|-----|------|------|
| `this` | 現在のスコープ | all | various |
| `root` | 元のコンテキスト（例: country_eventのcountry） | all | various |
| `from` | 前のイベントのROOTまたはハードコードされたFROM | all | various |
| `prev` | 一つ前のスコープ | all | various |
| `prevprev` | 二つ前のスコープ | all | various |
| `prevprevprev` | 三つ前のスコープ | all | various |
| `prevprevprevprev` | 四つ前のスコープ | all | various |
| `fromfrom` | FROMのFROM | all | various |
| `fromfromfrom` | FROMFROMのFROM | all | various |
| `fromfromfromfrom` | FROMFROMFROMのFROM | all | various |
| `no_scope` | 空のスコープ | all | no_scope |
