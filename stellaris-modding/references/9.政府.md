# 政府 (Governments)

## 概要

政府 (Governments) は、帝国の政治的風味を決定するシステムです。志向、統治形態、国是の組み合わせに基づいて自動的に決定され、支配者や後継者の称号などに影響します。政府のMOD関連ファイルは以下のディレクトリに配置されます：

| カテゴリ | パス |
|:--|:--|
| 政府形態 | `common/governments/*.txt` |
| 統治形態 | `common/governments/authorities/*.txt` |
| 国是・起源 | `common/governments/civics/*.txt` |
| 評議員 | `common/governments/councilors/*.txt` |

---

## 政府形態 (Government Types)

政府形態は、帝国の形式的な政府タイプ（例：代議制民主主義、神聖王国など）を定義します。志向、統治形態、国是に基づいて自動的に選択されます。

```pdx
# /common/governments/my_governments.txt

my_government = {
    ruler_title = LOC_MY_RULER_TITLE           # 支配者の称号
    ruler_title_female = LOC_MY_RULER_TITLE_F  # 女性支配者の称号
    
    heir_title = LOC_MY_HEIR_TITLE             # 後継者の称号
    heir_title_female = LOC_MY_HEIR_TITLE_F    # 女性後継者の称号
    
    use_regnal_names = yes                      # 王朝番号を使用
    dynastic_last_names = yes                   # 王朝姓を継承
    
    should_force_rename = no                    # 政府移行時に帝国名をランダム化するか（AIのみ）
    
    leader_class = official                     # 選挙時、敗れた支配者が変換されるリーダークラス
    
    possible = {                                # この政府が適用される条件
        has_authority = auth_imperial
        has_ethic = ethic_spiritualist
        has_valid_civic = civic_imperial_cult
    }
    
    weight = 100                                # 複数条件を満たす場合の優先度
    
    election_candidates = {                     # 選挙候補者の重み付け（選挙制のみ）
        modifier = {
            add = 100
            is_ruler = yes
        }
    }
}
```

### 主要パラメータ

| キー | 説明 |
|:--|:--|
| `ruler_title` / `ruler_title_female` | 支配者の称号（ローカライゼーションキー） |
| `heir_title` / `heir_title_female` | 後継者の称号（帝政のみ） |
| `use_regnal_names` | 王朝番号（「〇〇世」）を使用 |
| `dynastic_last_names` | 後継者が支配者の姓を継承 |
| `should_force_rename` | 政府刷新時に帝国名をランダム化（AI限定） |
| `leader_class` | 選挙敗北した支配者の変換先クラス |
| `possible` | 条件式。Countryスコープで評価 |
| `weight` | 優先度。高いほど選ばれやすい |
| `election_candidates` | 選挙候補者の重み付け |

## 統治形態 (Authorities)

統治形態は、帝国の権力分配の仕組みを決定します。選挙制・世襲制・不死の支配者など、根本的なプレイスタイルに影響します。

```pdx
# /common/governments/authorities/my_authorities.txt

auth_my_authority = {
    # --- 選挙システム ---
    election_type = democratic                 # democratic / oligarchic / none
    election_term_years = 10                   # 選挙周期（年）
    election_term_variance = 2                 # 選挙周期のランダム幅
    re_election_allowed = yes                  # 再選可能か
    can_have_emergency_elections = no          # 緊急選挙が可能か
    emergency_election_cost = 250              # 緊急選挙のコスト（影響力）
    max_election_candidates = -1               # 候補者の最大数（-1で無制限）
    
    # --- 世襲システム ---
    has_heir = no                              # 後継者システムを使用
    
    # --- Mandate/Agenda ---
    uses_mandates = yes                        # マンデートシステム（民主制）
    has_agendas = no                           # アジェンダシステム
    
    # --- 評議員 ---
    ruler_council_position = councilor_ruler_democratic
    
    # --- 視覚効果 ---
    color = { 81 140 44 255 }                  # UIカラー（RGBA）
    
    # --- 追加設定 ---
    has_factions = yes                         # 派閥システムを使用
    can_reform = yes                           # 政府刷新でこの形態に変更可能か
    valid_for_released_vassal = yes            # 解放した従属国で選択可能か
    
    # --- 種族特性 ---
    traits = {
        trait = trait_hive_mind                # 開始時に付与する特性
    }
    
    # --- ローカライゼーション ---
    localization_postfix = hive_mind           # ローカライズキーの接尾辞
    trait_localization_postfix = _hive
    
    # --- 条件 ---
    playable = {                               # プレイヤーが選択可能な条件
        has_utopia = yes
    }
    possible = {                               # この統治形態が適用可能な条件
        ethics = {
            value = ethic_gestalt_consciousness
        }
    }
    random_weight = {
        base = 2
    }
    
    # --- 修正値 ---
    country_modifier = {
        faction_approval = 0.10
        country_leader_pool_size = 1
    }
    
    # --- 説明タグ ---
    tags = {
        AUTHORITY_SUCCESSION_TYPE
        AUTHORITY_DEMOCRATIC_SUCCESSION
    }
    
    # --- 上位形態への変換 ---
    advanced_authority_swap = {
        name = "auth_cyber_creed_democratic"
        description = "auth_cyber_creed_democratic_desc"
        inherit_icon = no
        inherit_effects = no
        ruler_council_position = councilor_ruler_democratic_cyber
        
        trigger = {
            has_country_flag = cyber_creed_advanced_government
        }
        modifier = {
            pop_ethic_spiritualist_attraction_mult = 0.2
        }
        tags = {
            AUTHORITY_SUCCESSION_TYPE
        }
    }
}
```

### 統治形態のカテゴリ

| 統治形態 | 選挙 | 後継者 | 派閥 | 備考 |
|:--|:--:|:--:|:--:|:--|
| 民主制 (`auth_democratic`) | ✓ | ✗ | ✓ | マンデート、再選可能 |
| 寡頭制 (`auth_oligarchic`) | ✓ | ✗ | ✓ | アジェンダ、緊急選挙可能 |
| 独裁制 (`auth_dictatorial`) | △ | ✗ | ✓ | 終身制、死後に選挙 |
| 帝政 (`auth_imperial`) | ✗ | ✓ | ✓ | 後継者システム |
| 集合意識 (`auth_hive_mind`) | ✗ | ✗ | ✗ | 不死の支配者 |
| 機械知性 (`auth_machine_intelligence`) | ✗ | ✗ | ✗ | 不死の支配者 |

### 上位形態への変換 (advanced_authority_swap)

アセンションなどで統治形態が進化した際に適用される上位形態です。

| キー | 説明 |
|:--|:--|
| `name` | 上位形態の名前キー |
| `description` | 説明文のローカライズキー |
| `inherit_icon` | 元の統治形態のアイコンを継承するか |
| `inherit_effects` | 元の統治形態の`modifier`を継承するか |
| `trigger` | 適用条件（Countryスコープ） |
| `modifier` | 上位形態の修正値 |
| `tags` | 説明タグ |
| `has_heir` / `election_type` など | 選挙/継承システムを上書き可能 |

### 注意

- `government` 側の `possible` は `has_ethic` / `has_authority` / `has_valid_civic` を前提に組む。他のトリガーは未定義挙動になりやすい。
- `authority` 側の `random_weight` は no scope で評価される。帝国状態を見たい場合は `possible` / `playable` / `advanced_authority_swap.trigger` 側へ寄せる。
- エフェクトで authority swap を発火させた後は `advanced_authority_refresh = yes` を併用し、称号、評議員席、modifier の再計算を明示する。
- `advanced_authority_swap` で `ruler_council_position` を変える場合、対応する `common/governments/councilors/*.txt` とローカライズを同時に用意する。

## 国是 (Civics) と起源 (Origins)

国是は帝国に追加のボーナスやプレイスタイル変更を与えます。起源は `is_origin = yes` を持つ特殊な国是で、ゲーム開始条件を決定します。

```pdx
# /common/governments/civics/my_civics.txt

civic_my_civic = {
    # --- 基本設定 ---
    cost = 1                                   # 国是ポイントコスト（デフォルト1）
    
    # --- アイコン ---
    icon = "gfx/interface/icons/governments/civics/my_icon.dds"
    
    # --- 条件 ---
    playable = {                               # プレイヤー選択可能条件（グローバル）
        host_has_dlc = "Utopia"
    }
    potential = {                              # 表示条件（特殊構文）
        ethics = { NOT = { value = ethic_gestalt_consciousness } }
        authority = { NOT = { value = auth_corporate } }
    }
    possible = {                               # 選択可能条件（特殊構文）
        ethics = {
            OR = {
                text = civic_tooltip_spiritualist
                value = ethic_spiritualist
                value = ethic_fanatic_spiritualist
            }
        }
        civics = {
            NOT = { value = civic_incompatible }
        }
    }
    
    # --- 変更制限 ---
    modification = yes                         # ゲーム中に追加/削除可能
    pickable_at_start = yes                   # 帝国作成時に選択可能
    
    # --- AI ---
    random_weight = { base = 5 }
    ai_weight = {
        base = 100
        modifier = {
            factor = 2
            has_ai_personality = evangelising_zealots
        }
    }
    
    # --- 効果 ---
    description = "civic_my_tooltip"           # カスタム説明文
    modifier = {
        pop_ethic_spiritualist_attraction_mult = 0.15
    }
    
    # --- 種族特性 ---
    traits = {
        trait = trait_cybernetic
    }
    
    # --- 条件付き名前変更 ---
    swap_type = {
        name = civic_my_civic_lithoid
        description = "civic_my_civic_lithoid_effects"
        trigger = {
            local_human_species_class = LITHOID
        }
    }
}
```

### 起源固有のパラメータ

```pdx
origin_my_origin = {
    is_origin = yes                            # 起源として機能
    icon = "gfx/interface/icons/origins/origins_my_origin.dds"
    picture = GFX_origin_my_origin             # 帝国作成画面の背景画像
    
    # --- 開始惑星 ---
    starting_colony = pc_gaia                  # 開始惑星タイプの上書き
    habitability_preference = pc_gaia          # 気候適性特性の上書き
    preferred_planet_class_neighbor = no       # 保証された居住可能惑星のタイプを一致させない
    non_colonizable_planet_class_neighbor = no # 保証された居住可能惑星を無効化
    
    # --- ゲーム開始 ---
    initializers = { my_system_init }          # 使用する星系初期化子
    flags = { my_country_flag }                # 開始時に設定するフラグ
    advanced_start = yes                       # 先進スタートを許可
    
    # --- 二次種族 ---
    has_secondary_species = {
        title = LOC_SECONDARY_SPECIES
        traits = {
            trait = trait_syncretic_proles
        }
    }
}
```

### 主要パラメータ（国是・起源共通）

| キー | 国是 | 起源 | 説明 |
|:--|:--:|:--:|:--|
| `is_origin` | ✗ | ✓ | 起源として定義 |
| `playable` | ✓ | ✓ | プレイヤー選択可能条件 |
| `potential` | ✓ | ✓ | 表示条件（特殊構文） |
| `possible` | ✓ | ✓ | 選択可能条件（特殊構文） |
| `modification` | ✓ | ✗ | ゲーム中に変更可能か |
| `pickable_at_start` | ✓ | ✗ | 帝国作成時に選択可能か |
| `random_weight` | ✓ | ✓ | ランダム選択時の重み |
| `modifier` | ✓ | ✓ | 適用する修正値 |
| `traits` | ✓ | ✓ | 開始時に付与する種族特性 |
| `has_secondary_species` | ✓ | ✓ | 二次種族の定義 |
| `swap_type` | ✓ | ✗ | 条件付きの名前/効果変更 |

## 特殊条件構文

国是・起源の `potential` と `possible` ブロックでは、通常のConditionsではなく特殊な構文を使用します。これはパフォーマンス上の理由によるもので、ゲーム中に全ての帝国の全国是を毎日チェックするのを避けるためです。

```pdx
possible = {
    authority = {
        OR = {
            text = civic_tooltip_dic_imp       # ツールチップ代替テキスト
            value = auth_dictatorial
            value = auth_imperial
        }
    }
    ethics = {
        value = ethic_fanatic_xenophobe
        OR = {
            value = ethic_militarist
            value = ethic_spiritualist
        }
        NOR = {
            text = civic_tooltip_not_pacifist
            value = ethic_pacifist
            value = ethic_fanatic_pacifist
        }
    }
    origin = {
        NOR = {
            value = origin_common_ground
            value = origin_hegemon
        }
    }
    civics = {
        NOT = { value = civic_barbaric_despoilers }
    }
    species_archetype = {
        value = LITHOID
    }
}
```

### 利用可能なカテゴリ

| カテゴリ | 説明 |
|:--|:--|
| `authority` | 統治形態をチェック |
| `ethics` | 志向をチェック |
| `origin` | 起源をチェック |
| `civics` | 他の国是をチェック |
| `species_archetype` | 種族原型をチェック（BIOLOGICAL, MACHINE, LITHOID など） |
| `country_type` | 国タイプをチェック |

### 論理演算子

- `NOT`, `AND`, `OR`, `NOR` が使用可能
- `text = <loc_key>` でツールチップを置換可能
- **ネストした論理演算子は使用不可**
- **カテゴリをまたぐ論理演算子は使用不可**

> [!CAUTION]
> 以下のパターンは**動作しません**：
> ```pdx
> # ネストした論理演算子 ❌
> ethics = {
>     OR = {
>         AND = {
>             value = ethic_fanatic_authoritarian
>             value = ethic_militarist
>         }
>     }
> }
> 
> # カテゴリをまたぐOR ❌
> OR = {
>     ethics = { value = ethic_materialist }
>     authority = { value = auth_imperial }
> }
> ```

## 評議員 (Councilors)

評議員は帝国の行政評議会のメンバーを定義します。各評議員ポジションには特定のスキルやボーナスが関連付けられます。

```pdx
# /common/governments/councilors/my_councilors.txt

councilor_my_position = {
    leader_class = { official scientist }      # 対応するリーダークラス
    
    is_leader_possible = {                     # リーダーがこのポジションに就ける条件
        NOT = { has_trait = leader_trait_corrupt }
    }
    
    modifier = {                               # このポジションがもたらす修正値
        category_biology_research_speed_mult = 0.02    # スキルあたりの倍率
    }
    
    custom_tooltip = councilor_my_position_tt
}
```

## 条件・エフェクト

### 統治形態のチェック

```pdx
has_authority = auth_democratic               # 特定の統治形態か
is_gestalt = yes                              # 集合意識か
is_machine_empire = yes                       # 機械帝国か
is_hive_empire = yes                          # ハイブ帝国か
```

### 国是のチェック

```pdx
has_civic = civic_fanatic_purifiers          # 国是を持つか（無効化されていても）
has_valid_civic = civic_fanatic_purifiers    # 有効な国是を持つか
```

### 起源のチェック

```pdx
has_origin = origin_syncretic_evolution       # 特定の起源か
```

### 政府形態のチェック

```pdx
has_government = gov_military_dictatorship    # 特定の政府形態か
```

## ベストプラクティス

### ファイル構成

- 政府形態は `common/governments/` に数字プレフィックス付きで配置
- 国是は `common/governments/civics/` に配置
- ファイル名プレフィックスによる上書きは前提にしない。差し替えが必要なら対象システムのバニラ実装を先に確認

### パフォーマンス

- `potential` と `possible` は軽量な特殊構文を使用
- 複雑なチェックはイベントやdecisionで実装

### 互換性

- `has_valid_civic` を使用して無効化された国是を除外
- `modification = no` で変更不可の国是を適切にマーク
- DLC固有の機能は `playable` で制限
