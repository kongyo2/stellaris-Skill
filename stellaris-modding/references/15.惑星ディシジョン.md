# 惑星ディシジョン (Decisions)

惑星ディシジョンは、惑星上で実行可能なアクションを定義する。`common/decisions/xxx.txt`に配置。

---

## 基本構造

```pdx
decision_example = {
    owned_planets_only = yes        # 植民済み惑星のみ (yes) / 領内の未植民惑星も可 (no)
    icon = decision_icon_name       # gfx/interface/icons/decisions/ 配下のアイコン
    sound = event_sound_name        # 実行時のサウンドエフェクト
    
    enactment_time = <int>          # 実行完了までの日数（0=即時実行）
    prerequisites = { "tech_xxx" }  # 必要テクノロジーリスト
    
    resources = {                   # コスト定義
        category = decisions
        cost = {
            energy = 1000
            influence = 50
        }
    }
    
    potential = { }                 # 表示条件（惑星スコープ）
    allow = { }                     # 実行可能条件（惑星スコープ）
    effect = { }                    # 実行時エフェクト（惑星スコープ）
    
    ai_weight = {                   # AIの選択重み付け
        weight = 1
        modifier = { }
    }
}
```

---

## 主要フィールド

| フィールド | 型 | 説明 |
|-----------|-----|------|
| `owned_planets_only` | bool | `yes`=植民惑星のみ、`no`=領内未植民惑星でも可 |
| `icon` | string | GFXエントリまたはファイル名（パス・拡張子不要） |
| `sound` | string | サウンドエフェクトエントリ |
| `enactment_time` | int | 実行完了日数。0=即時、>0=建設キュー経由 |
| `prerequisites` | list | 必要テクノロジーのリスト |
| `resources` | block | [Economy Unit]形式のコスト定義 |
| `potential` | block | 表示条件（条件を満たさない場合リストに表示されない） |
| `allow` | block | 実行可能条件（条件を満たさない場合グレーアウト） |
| `effect` | block | 実行時エフェクト |
| `ai_weight` | block | AIの選択傾向 |

> [!IMPORTANT]
> Decisionsには**modifier**フィールドが存在しない。惑星にModifierを追加する場合は`effect`ブロック内で`add_modifier`を使用する。

---

## enactment_timeの挙動

| 値 | 挙動 |
|----|------|
| `0`または未指定 | 即時実行。建設キューを経由しない |
| `> 0` | 建設キューに追加。完了まで待機が必要。キャンセル可能 |

> [!WARNING]
> `owned_planets_only = no`の場合、未植民惑星には建設キューUIが存在しないため、`enactment_time = 0`を推奨。進捗確認やキャンセルができなくなる。

---

## 条件付きコスト

`resources`ブロック内で`trigger`を使用し、条件に応じたコストを定義可能：

```pdx
resources = {
    category = decisions
    cost = {
        trigger = { pop_amount < 1000 }
        consumer_goods = 100
    }
    cost = {
        trigger = { pop_amount >= 1000 }
        consumer_goods = 200
    }
}
```

## バニラ例：リソースブースト

```pdx
decision_planet_luxuries_boost = {
    owned_planets_only = yes
    sound = event_administrative_work
    icon = decision_luxuries
    
    resources = {
        category = decisions
        cost = {
            trigger = { pop_amount < 1000 }
            consumer_goods = 100
        }
        cost = {
            trigger = { pop_amount >= 1000 }
            consumer_goods = 200
        }
    }
    
    potential = {
        owner = { is_regular_empire = yes }
    }
    
    allow = {
        NOT = { has_modifier = planet_luxuries_boost }
    }
    
    effect = {
        add_modifier = { modifier = "planet_luxuries_boost" days = 3600 }
    }
    
    ai_weight = {
        weight = 1
        modifier = {
            add = 10
            free_amenities < -5
            owner = {
                resource_stockpile_compare = {
                    resource = consumer_goods
                    value > 300
                }
            }
        }
    }
}
```

## バニラ例：特殊惑星タイプ（流刑地）

```pdx
decision_penal_colony = {
    owned_planets_only = yes
    enactment_time = 180
    
    prerequisites = { "tech_penal_colonies" }
    
    resources = {
        category = decisions
        cost = { unity = 1000 }
    }
    
    potential = {
        is_capital = no
        is_artificial = no
        NOR = {
            has_modifier = resort_colony
            has_modifier = slave_colony
            is_planet_class = pc_city
        }
        owner = { is_regular_empire = yes }
        NOT = {
            owner = {
                any_owned_planet = { has_modifier = penal_colony }
            }
        }
    }
    
    effect = {
        add_modifier = {
            modifier = penal_colony
            days = -1  # 永続
        }
        hidden_effect = {
            validate_planet_buildings_and_districts = yes
        }
    }
    
    ai_weight = { weight = 0 }
}
```

## バニラ例：未植民惑星向け

```pdx
decision_consecrated_worlds = {
    # owned_planets_only = yesを指定しない = デフォルトでno相当
    
    resources = {
        category = decisions
        cost = { unity = 500 }
    }
    
    potential = {
        exists = from
        from = { has_ascension_perk = ap_consecrated_worlds }
        exists = space_owner
        space_owner = { is_same_value = from }
        is_star = no
        is_artificial = no
        NOT = { has_planet_flag = consecrated@from }
    }
    
    allow = {
        custom_tooltip = {
            from = {
                check_variable = {
                    which = num_consecrated_worlds
                    value < 3
                }
            }
            fail_text = consecrated_worlds_too_many
        }
        custom_tooltip = {
            NOT = { exists = owner }
            fail_text = consecrated_worlds_planet_owned
        }
    }
    
    effect = {
        custom_tooltip = decision_consecrated_worlds_effects_custom
        hidden_effect = {
            set_planet_flag = consecrated@from
            # 惑星タイプに応じたModifier付与ロジック...
        }
    }
}
```

## モディファイアの追加・除去パターン

ディシジョンは直接`modifier`を持てないため、Static Modifierを操作する：

```pdx
# 追加
effect = {
    add_modifier = {
        modifier = modifier_name
        days = 3600      # 期間指定（日数）
    }
}

effect = {
    add_modifier = {
        modifier = modifier_name
        days = -1        # 永続
    }
}

# 除去
effect = {
    remove_modifier = modifier_name
}
```

## AI Weightパターン

```pdx
ai_weight = {
    weight = 5            # 基本重み
    
    # 条件で重みを0にする（実行させない）
    modifier = {
        factor = 0
        free_housing > 0
    }
    
    # 条件で重みを加算
    modifier = {
        add = 10
        free_amenities < -5
    }
    
    # オーナーの状態をチェック
    modifier = {
        factor = 0
        owner = {
            resource_stockpile_compare = {
                resource = energy
                value < 1000
            }
        }
    }
}
```

## 有用なトリガー（潜在条件・許可条件）

| トリガー | 説明 |
|---------|------|
| `is_capital` | 首都惑星か |
| `is_artificial` | 人工惑星（ハビタット、リングワールド等）か |
| `is_planet_class = pc_xxx` | 惑星タイプ |
| `has_modifier = xxx` | 特定Modifierを持っているか |
| `has_building = xxx` | 特定建造物があるか |
| `has_designation = col_xxx` | 惑星指定タイプ |
| `free_housing` | 空き住居数 |
| `free_amenities` | 空きアメニティ |
| `pop_amount` | POP（実際は人口×100） |
| `num_pops` | POP数 |
| `num_districts` | ディストリクト数 |
| `has_building_construction` | 建設中か |
| `owner = { }` | オーナー帝国スコープ |

---

## 有用なエフェクト

| エフェクト | 説明 |
|-----------|------|
| `add_modifier` | Static Modifierを追加 |
| `remove_modifier` | Static Modifierを除去 |
| `set_planet_flag` | 惑星フラグをセット |
| `remove_planet_flag` | 惑星フラグを除去 |
| `add_deposit` | 地形を追加 |
| `remove_deposit` | 地形を除去 |
| `add_building` | 建造物を追加 |
| `remove_building` | 建造物を除去 |
| `validate_planet_buildings_and_districts` | 建造物・ディストリクトの再検証 |
| `custom_tooltip` | カスタムツールチップ表示 |
| `hidden_effect` | UI非表示でエフェクト実行 |
