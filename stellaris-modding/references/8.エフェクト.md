# エフェクト (Effects)

## 概要

Stellarisのスクリプトにおける「Effect（エフェクト）」は、ゲームの状態を能動的に変更するための命令です。これは他のプログラミング言語における「関数」や「手続き」に似た役割を果たします。

- **トリガー（Trigger）**: 条件が真(true)か偽(false)かを評価
- **エフェクト（Effect）**: 実際にゲーム内で何かを実行

エフェクトは、呼び出されたスコープに対して実行されます。補正（Modifier）の追加、オブジェクト（国家、艦船、POPなど）の生成や破壊、変数の値の変更といった操作はすべてエフェクトによって行われます。

---

## エフェクトの基本構文

エフェクトは `効果名 = 値` または `効果名 = { パラメータ }` の形式で記述します：

```pdx
# 単純なエフェクト
add_resource = { energy = 100 }

# 条件付きエフェクト（ifブロック内）
if = {
    limit = { has_ethic = ethic_militarist }
    add_resource = { alloys = 500 }
}

# スコープ切り替えとエフェクト
every_owned_planet = {
    limit = { is_capital = no }
    add_modifier = { modifier = resource_bonus_modifier days = 360 }
}
```

---

## 論理・制御フロー

スクリプトの実行フローを制御するエフェクトです。

| エフェクト | 説明 | 使用例 |
|:--|:--|:--|
| `if` | `limit`ブロック内のトリガーが真の場合に実行 | `if = { limit = { <triggers> } <effects> }` |
| `else_if` | 先行する`if`が偽で、自身の`limit`が真の場合に実行 | `else_if = { limit = { <triggers> } <effects> }` |
| `else` | 先行する条件がすべて偽だった場合に実行 | `else = { <effects> }` |
| `switch` | `trigger`の値に応じて一致するキーを実行 | `switch = { trigger = <trigger> value1 = { ... } }` |
| `random_list` | 重み付けに基づいてランダムに一つを実行 | `random_list = { 50 = { ... } 25 = { ... } }` |
| `random` | 確率に基づいてエフェクトを実行 | `random = { chance = 50 <effects> }` |
| `while` | 条件が真である限り繰り返し実行 | `while = { limit = { <triggers> } count = 10 <effects> }` |
| `break` | 現在のエフェクトブロックを中断 | `break = yes` |
| `hidden_effect` | ツールチップに表示せずに実行 | `hidden_effect = { <effects> }` |
| `custom_tooltip` | 任意のテキストをツールチップに表示 | `custom_tooltip = "my_tooltip_loc_key"` |

### 条件分岐の例

```pdx
if = {
    limit = {
        has_ethic = ethic_materialist
        num_owned_planets >= 5
    }
    add_resource = { engineering_research = 1000 }
}
else_if = {
    limit = { has_ethic = ethic_spiritualist }
    add_resource = { society_research = 1000 }
}
else = {
    add_resource = { physics_research = 500 }
}
```

### random_listの例

```pdx
random_list = {
    50 = {                           # 50%の確率
        add_resource = { minerals = 100 }
    }
    30 = {                           # 30%の確率
        modifier = {                 # 条件付き重み調整
            factor = 2
            has_technology = tech_mining
        }
        add_resource = { minerals = 200 }
    }
    20 = {                           # 20%の確率
        # 何もしない
    }
}
```

## オブジェクトの生成・破壊

新しいオブジェクトを生成したり、既存のものを削除するエフェクトです。

### 国・リーダー

| エフェクト | スコープ | 説明 |
|:--|:--|:--|
| `create_country` | all | 新しい国家を生成 |
| `destroy_country` | country | 国家を破壊 |
| `create_leader` | country | 新しいリーダーを生成 |
| `kill_leader` | leader | リーダーを死亡させる |
| `exile_leader_as` | leader | リーダーを追放し変数に保存 |
| `unexile_leader` | country | 追放したリーダーを呼び戻す |

```pdx
# 国家の生成
create_country = {
    name = "NAME_Pirate_Confederation"
    type = pirate
    species = random
    flag = {
        background = { category = "backgrounds" file = "00_solid.dds" }
        colors = { "black" "red" "null" "null" }
    }
    effect = {
        save_global_event_target_as = pirate_country
    }
}

# リーダーの生成
create_leader = {
    class = scientist
    species = owner_main_species
    name = random
    skill = 5
    traits = {
        trait = leader_trait_expertise_biology
    }
    effect = {
        save_event_target_as = new_scientist
    }
}
```

### 艦隊・艦船

| エフェクト | スコープ | 説明 |
|:--|:--|:--|
| `create_fleet` | all | 新しい艦隊を生成 |
| `create_ship` | fleet, starbase | 艦隊/宇宙基地に艦船を追加 |
| `destroy_fleet` | fleet | 艦隊を破壊 |
| `create_military_fleet` | country | 艦隊テンプレートから軍事艦隊を生成 |

```pdx
# 艦隊と艦船の生成
create_fleet = {
    name = "NAME_Exploration_Fleet"
    effect = {
        set_owner = root
        create_ship = {
            name = "NAME_Survey_Ship"
            design = "Explorer"
        }
        set_location = root.capital_scope
        save_event_target_as = new_fleet
    }
}
```

| エフェクト | スコープ | 説明 |
|:--|:--|:--|
| `create_colony` | planet | 惑星に入植地を生成 |
| `destroy_colony` | planet | 入植地を破壊 |
| `remove_planet` | planet | 惑星を完全に消滅 |
| `spawn_system` | galactic_object | 新しい星系を生成 |
| `spawn_planet` | galactic_object | 星系に惑星を生成 |
| `spawn_megastructure` | galactic_object | 巨大建造物を生成 |

```pdx

    owner = root
    species = root.species
    ethos = owner
}

# 星系の生成

    min_distance >= 20

}
```

## 状態変更

オブジェクトの所有権、クラス、名前などを変更するエフェクトです。

### 所有権・支配権

| エフェクト | スコープ | 説明 |
|:--|:--|:--|
| `set_owner` | planet, fleet, leader, army, starbase, megastructure | 所有者を変更 |
| `set_controller` | planet, fleet | 支配者を変更 |

```pdx
# 惑星の所有権移転
set_owner = event_target:new_owner

# 艦隊の支配権移転（戦争中など）
set_controller = from
```

### 惑星変更

| エフェクト | スコープ | 説明 |
|:--|:--|:--|
| `change_pc` | planet | 惑星クラスを変更 |
| `set_capital` | planet | 首都に設定 |
| `add_building` | planet | 建物を建設 |
| `remove_building` | planet | 建物を削除 |
| `add_district` | planet | 区画を追加 |
| `remove_district` | planet | 区画を削除 |
| `clear_blockers` | planet | すべてのブロッカーを除去 |
| `add_deposit` | planet | 資源堆積物を追加 |
| `remove_deposit` | planet | 資源堆積物を削除 |
| `reroll_deposits` | planet | 堆積物を再生成 |
| `reroll_planet` | planet | 惑星を再生成（堆積物含む） |
| `surveyed = { ... }` | planet | 調査状態を設定 |

```pdx
# テラフォーミング
change_pc = pc_gaia

# 建物の追加
add_building = building_research_lab_1

# 区画の追加
add_district = district_mining

# 調査済みに設定
set_surveyed = {
    surveyed = yes
    surveyor = root
}
```

### 補正（Modifier）

| エフェクト | スコープ | 説明 |
|:--|:--|:--|
| `add_modifier` | most | 補正を追加 |
| `remove_modifier` | most | 補正を削除 |
| `has_modifier` | most | 補正の有無を確認（トリガー） |

```pdx
# 永続的な補正
add_modifier = {
    modifier = prosperous_economy
}

# 期限付き補正
add_modifier = {
    modifier = resource_bonus
    days = 1800              # 5年間
}

# 乗数付き補正
add_modifier = {
    modifier = pop_happiness_bonus
    mult = trigger:num_owned_planets
}
```

### 名前変更

| エフェクト | スコープ | 説明 |
|:--|:--|:--|
| `set_name` | most | オブジェクトの名前を設定 |

```pdx
# 動的な名前設定
set_name = "NAME_Conquered_[Root.GetName]"
```

## 変数・フラグ・ターゲット

スクリプト内で後から参照するための変数やフラグを設定、変更、削除するエフェクトです。

### フラグ操作

| エフェクト | スコープ | 説明 |
|:--|:--|:--|
| `set_country_flag` | country | 国家フラグを設定 |
| `remove_country_flag` | country | 国家フラグを削除 |
| `set_planet_flag` | planet | 惑星フラグを設定 |
| `remove_planet_flag` | planet | 惑星フラグを削除 |
| `set_star_flag` | galactic_object | 星系フラグを設定 |
| `remove_star_flag` | galactic_object | 星系フラグを削除 |
| `set_fleet_flag` | fleet | 艦隊フラグを設定 |
| `remove_fleet_flag` | fleet | 艦隊フラグを削除 |
| `set_global_flag` | all | グローバルフラグを設定 |
| `remove_global_flag` | all | グローバルフラグを削除 |


### 変数操作

| エフェクト | スコープ | 説明 |
|:--|:--|:--|
| `set_variable` | most | 変数を設定 |
| `change_variable` | most | 変数を加算 |
| `subtract_variable` | most | 変数を減算 |
| `multiply_variable` | most | 変数を乗算 |
| `divide_variable` | most | 変数を除算 |
| `modulo_variable` | most | 剰余を計算 |
| `round_variable` | most | 最も近い整数に丸め |
| `floor_variable` | most | 切り捨て |
| `ceiling_variable` | most | 切り上げ |
| `clear_variable` | most | 変数をクリア |
| `export_trigger_value_to_variable` | most | トリガー値を変数にエクスポート |

```pdx

    which = my_counter

    which = my_counter
    value = 1

```

## ループ・反復処理

特定の条件を満たすオブジェクト群に対して同じエフェクトを繰り返し実行します。

| パターン | 説明 |
|:--|:--|
| `every_*` | 条件を満たすすべてのオブジェクトに実行 |
| `random_*` | 条件を満たすものからランダムに1つ選択 |
| `ordered_*` | ソートして指定順位のものに実行 |
| `any_*` | いずれかが条件を満たすかを判定（トリガー） |
| `count_*` | 条件を満たす数をカウント（トリガー） |

### 主要なイテレーター

| エフェクト | スコープ | 対象 |
|:--|:--|:--|
| `every_country` | all | すべての国家 |
| `random_country` | all | ランダムな国家 |
| `every_owned_planet` | country, sector | 所有惑星 |
| `random_owned_planet` | country, sector | ランダムな所有惑星 |
| `every_owned_pop` | planet, country, sector | 所有POP |
| `random_owned_pop` | planet, country, sector | ランダムなPOP |
| `every_owned_fleet` | country | 所有艦隊 |
| `random_owned_fleet` | country | ランダムな所有艦隊 |
| `every_fleet_in_system` | galactic_object | 星系内艦隊 |
| `every_owned_leader` | country | 所有リーダー |
| `every_system_planet` | galactic_object | 星系内惑星 |
| `every_neighbor_country` | country | 隣接国家 |
| `every_subject` | country | 従属国 |
| `every_federation_ally` | country | 連邦同盟国 |

```pdx
# すべての所有惑星に実行
every_owned_planet = {
    limit = {
        is_capital = no
        num_pops >= 10
    }
    add_modifier = { modifier = production_bonus days = 180 }
}

# ランダムな惑星を選択
random_owned_planet = {
    limit = { is_colonizable = yes }
    weights = {
        base = 1
        modifier = {
            add = 10
            has_modifier = lush_planet
        }
    }
    save_event_target_as = selected_planet
}

# ソートして上位を選択
ordered_owned_leader = {
    limit = { leader_class = scientist }
    order_by = trigger:skill_level
    position = 0                          # 最高スキルのリーダー
    add_trait = leader_trait_expertise_field_manipulation
}
```

## スクリプト化されたエフェクト (Scripted Effects)

同じエフェクトの組み合わせを複数の場所で再利用したい場合、**スクリプト化されたエフェクト**として定義できます。

### 定義場所

`common/scripted_effects/` ディレクトリに `.txt` ファイルとして配置します。

### 基本構文

```pdx
# /common/scripted_effects/my_effects.txt

# シンプルなエフェクト
my_simple_effect = {
    add_resource = { energy = 100 }
    add_modifier = { modifier = prosperity }
}

# パラメータ付きエフェクト
my_parameterized_effect = {
    add_resource = { $RESOURCE$ = $AMOUNT$ }
    set_country_flag = $FLAG$
}
```

### 使用方法

```pdx
# シンプルなエフェクトの呼び出し
immediate = {
    my_simple_effect = yes
}

# パラメータ付きエフェクトの呼び出し
immediate = {
    my_parameterized_effect = {
        RESOURCE = minerals
        AMOUNT = 500
        FLAG = received_minerals
    }
}
```

### 実際のスクリプト化されたエフェクトの例

バニラゲームから抜粋した実用的な例：

```pdx
# 惑星温度を上げる連鎖処理
increase_planet_temperature = {
    if = {
        limit = { is_planet_class = pc_frozen }
        change_pc = pc_arctic
        reroll_planet = yes
        clear_blockers = yes
    }
    else_if = {
        limit = { is_planet_class = pc_arctic }
        change_pc = pc_ocean
        reroll_planet = yes
        clear_blockers = yes
    }
    # ... 他の惑星タイプも同様
}

# デバッグ用スコープ出力
print_scope_effect = {
    log = "------------------------------"
    log = "- This: \\[This.GetName]"
    log = "- Root: \\[Root.GetName]"
    log = "- From: \\[From.GetName]"
    log = "- Owner: \\[Owner.GetName]"
    log = "------------------------------"
}

# サイボーグ種族への変更
create_cybernetic_species_mod = {
    species = {
        modify_species = {
            remove_trait = trait_latent_psionic
            remove_trait = trait_psionic
            effect = {
                create_species = {
                    is_mod = yes
                    name = this
                    class = this
                    portrait = this
                    traits = this
                    traits = {
                        ideal_planet_class = this
                        trait = trait_cybernetic
                    }
                }
            }
        }
    }
    owner = {
        every_owned_pop_group = {
            limit = { is_exact_same_species = event_target:changing_species }
            change_species = last_created_species
        }
    }
}
```

## よく使うパターン

### 安全なリソース消費

```pdx
if = {
```

```pdx
if = {
```

```pdx
while = {
```

## 注意事項

> [!IMPORTANT]
> エフェクトは呼び出されたスコープに対して実行されます。スクリプト化されたエフェクト内で`FROM`スコープを使用する際は、呼び出し元のコンテキストを常に意識してください。

> [!WARNING]
> `while`ループは注意して使用してください。`limit`を設定しない場合は必ず`count`を指定して無限ループを防止します。

> [!TIP]
> 複雑なエフェクトを開発するときは、`log`エフェクトで変数やスコープの値を出力してデバッグすると効率的です。
